#  Solar System Simulation (Three.js)

This project is an **interactive 3D solar system simulation** built using **Three.js**, featuring realistic planetary textures, normal-maps, adjustable orbital speeds, and an HDRI space background for immersive and realistic visuals.

##  Features

- Textured Planets with normal maps for added realism
- Saturn’s Rings modeled using `RingGeometry`
- HDRI Background for a deep-space atmosphere for realism
- Adjustable Orbital Speeds via slider controls
- Smooth Camera Controls with `OrbitControls`

##  Installation & Setup
### Prerequisites
- A modern web browser (Chrome, Firefox, Edge)
- A local web server i.e. LiveServer(to properly load textures)
### Run Locally
- Open this folder in VS Code
- Install the "Live Server" extension
- Right-click `index.html` → **"Open with Live Server"**
